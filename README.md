# 🤖 AI Mira Chatbot

A Gen-Z styled AI chatbot with humor, sass, and emotional intelligence.

## 🚀 Features
- Smart AI responses (Groq API)
- Typing animation UI
- Error handling + validation

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JS
- Backend: Node.js, Express
- AI: Groq (LLaMA 3)

## ⚙️ Run Locally

### Backend
cd backend
npm install
node server.js

### Frontend
Open frontend/index.html

## 🔐 Note
Add GROQ_API_KEY in .env file
